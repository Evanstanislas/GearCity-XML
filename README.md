**Gearcity XML Editor**

A Python based App that took a lot of inspiration from VENT's Gearcity's own ModTool. This project only has the AI XML implemented so far.

Check out the original app: https://github.com/gearcity/GearCity_ModTools
I would recommend to delve into those more as it has way more feature and capabilities than this app.
